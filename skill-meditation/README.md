# 🧘 skill-meditation (Reflection)

This module handles the nightly consolidation of memories. It reads the episodic daily memory (generated by `skill-memory`) and synthesizes it into the long-term cognitive summary (`MEMORY.md`), while keeping a log of how the agent's cognition evolves over time.

## Data Flow
- **Inputs**: `memory/YYYY-MM-DD.md`, `MEMORY.md`
- **Outputs**: Rewrites `MEMORY.md`, appends to `data/evolution.md`

## Zero Dependency
The meditation script uses pure standard Python `urllib` to make HTTP POST requests to any OpenAI-compatible endpoint. No `pip install openai` is required, ensuring absolute portability.

## Usage

You can run the meditation script manually or via cron (`skill-monitor`).

```bash
export LLM_API_BASE="https://api.openai.com/v1"
export LLM_API_KEY="sk-..."
export LLM_MODEL="gpt-4o"

python3 scripts/meditate.py --base-dir /path/to/agent/root
```

### Compatible with Any Local/Cloud LLM
Because it uses the standard OpenAI API shape, you can point it to:
- **Local Ollama**: `LLM_API_BASE="http://localhost:11434/v1" LLM_API_KEY="ollama" LLM_MODEL="llama3"`
- **LM Studio**: `LLM_API_BASE="http://localhost:1234/v1"`
- **DeepSeek / Gemini / Claude (via proxies)**.
