# 🪴 skill-topic-lab (Greenhouse / Farm)

This module acts as the lifecycle manager for the ideas (seeds) generated by `skill-daydream` or added manually. It evaluates the JSONL database of ideas, applying "daily decay" to neglected ideas and "planting" highly mature ideas by converting them into actionable GitHub Issues.

## Data Flow
- **Inputs**: `data/topic-lab-seeds.jsonl`
- **Outputs**: Rewrites `data/topic-lab-seeds.jsonl` (updates maturity, changes status to `planted` or `composted`). Interacts with the external environment via the `gh` CLI.

## Concept

Instead of a standard todo list where items sit forever, the Topic Lab treats ideas like plants:
- If you interact with them (e.g., watering them), their maturity increases.
- If you ignore them, their maturity drops by 5 points every `tick`.
- If maturity drops below 20, they become `composted` (archived).
- If maturity reaches 80+, they are automatically converted into a GitHub Issue and marked as `planted`.

## Usage

### 1. Run the Daily Tick (Decay & Harvest)
Normally triggered by `skill-monitor` once a day:
```bash
python3 scripts/farm.py --base-dir /path/to/agent/root --tick
```

### 2. Water a Seed (Increase Maturity)
If an idea is discussed in a chat, you can manually boost its maturity:
```bash
python3 scripts/farm.py --base-dir /path/to/agent/root --add-water "dd-a1b2c3d4"
```
